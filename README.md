# Splunk Security Labs

A hands-on Splunk home lab portfolio documenting the process of building, configuring, and using Splunk Enterprise in a cybersecurity environment.

This repository focuses on practical experience with Splunk, including installation, log collection, Windows event monitoring, search development, detection engineering, and security investigations. Each lab documents only completed work based on real testing in a dedicated home lab environment.

## Lab Environment

- Hyper-V
- Ubuntu Server 24.04.4 LTS
- Windows 10
- Windows Server 2022
- Splunk Enterprise
- Windows Universal Forwarder
- Sysmon
- PowerShell
- Windows Event Logs
